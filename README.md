## Quick Start

```bash
# Clone the repository
git clone https://github.com/WhiteMuush/sandevistan.git

# Navigate to the directory
cd sandevistan

# Make the script executable and run
chmod +x sandevistan.sh
sudo bash sandevistan.sh
```
